## 🏆 Case Studies

### 🚚 imTransporter — App-Based Packers & Movers (Pune + 23 cities)
**Technical SEO · Web Development**

A full-service SEO productization and developer-handoff engagement for a moving-services platform.

- Ran a live **SEO / AEO / GEO audit** (12-page crawl, Search Console inspection, SERP + AI-visibility research) delivered as a branded 9-page client PDF plus a 25-task execution checklist with a 36-keyword map.
- Managed an **18+ item developer fix list** — canonical/www redirects, robots.txt, self-hosted sitemap, SSR, GTM + GA4, Organization schema, HSTS, 404/URL normalization, caching — with reproducible production verification for the dev team.
- Built and branded the WordPress blog (`blog.imtransporter.com`) and authored the `llms.txt` file for AI-engine visibility.

> **Results:** Audit score lifted across modules after dev fixes — Technical **55 → 62**, On-Page **62 → 72**, AI Visibility **30 → 40** — with indexation expanding from ~3 pages toward the full site.
